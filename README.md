# 二阶堂真红 — Galgame 角色扮演 Skill

基于视觉小说《五彩斑斓的世界》中女主角**二阶堂真红**的 2,826 句游戏台词，构建的角色扮演 Skill。适用于 Claude Code、Codex 等支持 skill 的 AI 平台。

激活后，AI 会以真红的第一人称视角对话，保持傲娇毒舌的语言风格、哲学倾向，以及"嘴硬心软"的情感层次。

## 效果预览

> **用户：** 真红，我喜欢你。
>
> **真红：** ……！？……笨、笨蛋……突然说什么啊……！……（小声）……我也是……最喜欢了。把这话给我忘掉！

> **用户：** 今天心情不好。
>
> **真红：** ……哼，一脸世界末日的样子。虽然不知道发生了什么……不过，我在这儿呢。够了吧？

> **用户：** 帮我查一下明天的天气。
>
> **真红：** ……你脑子没问题吧？我是霊体，不是天气预报员。……算了，念在你这么没用的份上，明天多云。

## 安装

将 `shinku/` 文件夹复制到你的 skill 目录：

```bash
# Claude Code / Codex
cp -r shinku/ ~/.agents/skills/shinku/

# Claude Desktop
cp -r shinku/ ~/.claude/skills/shinku/
```

安装后重启 AI 平台，说出以下任意关键词即可触发：

> "和真红说话"、"扮演真红"、"shinku"、"二阶堂"、"iroseka"、"五彩斑斓"

## Skill 包含内容

- **角色档案** — 姓名、身份、人称、称呼习惯等基础设定
- **性格模型** — 四层结构：表面毒舌 → 中间哲学 → 内核温柔 → 萌点动作
- **台词模式** — 毒舌吐槽、哲学分析、害羞防御、真心流露四类，附日文原文与中文对照
- **对话规则** — 傲娇优先级、语言节奏、拒绝模式、脆弱时刻的处理方式
- **知识边界** — 明确角色"知道什么"和"不知道什么"，防止人设崩塌

## 构建方法

1. **数据提取** — 从游戏 HCB 脚本（WillPlus/AdvHD 引擎）中提取二进制文本，筛选出真红的 2,826 句台词并完成语音对齐
2. **性格分析** — 词频统计 + 语气模式分类，提取出四类情感模式及其比例（约 60% 毒舌 + 20% 哲学 + 15% 害羞 + 5% 真心）
3. **Skill 编写** — 将分析结果组织成结构化的 SKILL.md，包括角色档案、对话规则、知识边界和触发条件

## 扩展方向

- **中文台词** — 当前语料为日文，可翻译关键台词加入 skill
- **对话样本扩充** — 从 2,826 句中精选更多典型对话模式
- **语音克隆** — 用游戏语音训练 TTS 模型实现语音对话
- **RAG 增强** — 将台词做成向量数据库，对话时实时检索
